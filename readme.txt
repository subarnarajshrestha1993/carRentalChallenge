Instructions to Use the Code:

1. The script expects the input json file to be provided as an input. Full path of the input file has to be provided as input.
   eg: Enter the file path:sample_output.json
2. The output of the script is being saved into a json file (output.json), in the same folder where script is present
   The output json file will have
   a) id
   b) start_timestamp
   c) end_timestamp
   d) start_comments
   f) end_comments
   g) over_time (if the car was rented for more than 4 hours)
   h) car_damaged (if end event has comment)




 The script can be run for larger data file as well.
